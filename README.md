# API de Gestión de Películas y Usuarios

Este proyecto es una API que permite la gestión de películas y usuarios. Está construida con Node.js, Express y utiliza una base de datos para almacenar la información.
pasos :

Clona el repositorio.
Instala las dependencias.
Asegúrate de configurar tu base de datos en el archivo correspondiente en la carpeta /db.

Contribuir
Las contribuciones son bienvenidas. Si deseas contribuir, por favor haz un fork del repositorio y crea una nueva rama para tus cambios. Luego, envía un pull request con una descripción detallada de los cambios propuestos.
